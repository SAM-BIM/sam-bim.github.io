Overview

What is SAM?

SAM (Sustainable Analytical Model) is an open‑source toolkit for building analytical models and running energy and environmental studies. You can import or generate geometry, assign constructions and loads, configure systems, and run simulations — then review and export results.

Model creation

Create models from Revit / IFC / gbXML, or generate and edit directly in Rhino/Grasshopper.

Simulation & analysis

Use SAM’s engines and connectors to predict performance and compare design options.

Extensible

Easily extend with plugins (e.g., Solar, Psychrometrics, Mollier, Tas, OpenStudio, Ladybug Tools).

Basics

Core concepts

  • Single source of truth: geometry, assumptions, and results live together for transparent QA.
  • Two object families: Analytical (Panels, Spaces, Adjacency Cluster, Analytical Model) and Geometry (Face3D, edges, shells).
  • SI units: SAM uses SI throughout (temperature expressed in °C).
  • Montserratoperability: push/pull with Revit, Rhino/Grasshopper; exchange via IFC, gbXML, Excel; connect to Honeybee, Tas, Topologic, BHoM and more.
Objects

Analytical hierarchy

Panel
Wall / Floor / Roof element incl. type, apertures, construction.
Adjacency Cluster
Relationships between Panels and Spaces.
Analytical Model
Holds Panels, Spaces, Location, Materials, Profiles, assumptions.
Geometry
Face3D with external / internal edges; shells delineate space boundaries.

Get started

Install

  1. Grab the latest SAM installer.
  2. Follow the prompts to add SAM to your environment.
  3. Optionally build from source in Visual Studio using the SAM repository.

Tested with Rhino 6/7; Revit support varies by release — see installer notes.<\/p>

Workflow

Typical steps

  1. Import or create project geometry.
  2. Define constructions, loads, profiles, and systems.
  3. Build the Adjacency Cluster and validate the Analytical Model.
  4. Run simulations; compare scenarios; document assumptions & results.
  5. Exchange via IFC/gbXML or sync back to Revit.
